jQuery SlideShow

Default settings

$("#slide1").slideShow({page: '.slide'});

Drop effect with pager

$("#slide2").slideShow({page: '.slide', effect: 'drop', pager: true});

Slide effect with arrows

$("#slide3").slideShow({page: '.slide', effect: 'slide', arrows: true});

No js effect - pure CSS

$("#slide4").slideShow({page: '.slide', effect: 'toggle', classActive: 'active', classOn: 'slide-show', classOff: 'slide-hide'});
